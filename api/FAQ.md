# FAQ для сервиса booking

Вопросы, которые могут возникнуть

## Может ли клиент иметь несколько записей?
Нет, у клиента может быть только одна активная запись. Чтобы создать новую, нужно удалить существующую.

## Как работает расписание?
Day - структура для одного рабочего дня. Содержит рабочие часы и приоритет на половину дня. 
- Для клиентов:
    При выборе даты будет выдаваться соответствующий Day. Алгоритм рассчитывает свободные слоты.
- Для барберов: 
    Фронт должен позволять заполнить сразу несколько Day на одной странице.
